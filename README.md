# 🎵 Song AI Explorer

**Song AI Explorer** is a sleek Chrome extension designed for music enthusiasts and language learners. It uses Large Language Models (LLMs) to provide instant deep-dive analysis, translations, and cultural context for song lyrics directly within your browser.

## ✨ Features

  * **One-Click Analysis**: Highlight any text on a webpage and get an instant breakdown of the song's meaning.
  * **Context-Aware Chat**: After the initial analysis, engage in a continuous dialogue with the AI to ask follow-up questions about the artist, genre, or specific metaphors.
  * **Intelligent Text Retrieval**: Automatically detects lyrics on popular sites or uses your manual selection for precision.
  * **Cute & Modern UI**: A soft-themed, responsive chat interface that stays hidden until you need it, keeping your browsing experience clean.

## 🚀 Technical Stack

  * **Frontend**: HTML5, CSS3 (Custom animations), and Vanilla JavaScript.
  * **Backend logic**: Chrome Extension API (Manifest V3) with background service workers for asynchronous API handling.
  * **AI Integration**: Powered by OpenRouter API (DeepSeek models) for high-performance natural language processing.

## 🛠️ Installation

1.  Clone this repository or download the source files.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked** and select the extension folder.
5.  Open `background.js` and replace `API_KEY` with your personal OpenRouter key.

## 📖 Usage

1.  Navigate to any song lyric website.
2.  **Highlight** the lyrics you want to explore.
3.  Click the **Song AI Explorer** icon in your toolbar.
4.  Hit **✨ Analyze Lyrics** to start the magic\!
5.  Use the chat box to dive deeper into the song's "hidden" story.
